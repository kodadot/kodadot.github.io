# VueJS utilities for interaction with Substrate Networks (Kusama/Polkadot)

This project is in archive mode.

Only API & Settings packages are maintained.

Original team has forked this repository and working on Kusama NFT gallery - KodaDot and experimental NFT landscape - Metaprime.network

Feel free to join their Discord - KodaDot NFT explorers & MetaPrime community

You can check products build thanks to this libraries

  • nft.kodadot.xyz
  • metaprime.network

This is a collection of Vue.js libraries, utlities and classes that allows for easier use of the Polkadot JS infrastructure inside your apps from Vue apps.

💡 Example - Vue Identicon

In a rush and just want examples? Jump right in and get a handle the libraries in your projects.
