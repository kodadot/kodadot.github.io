# VueJS utilities for interaction with Substrate Networks (Kusama/Polkadot)

This project is in archive mode.

Only API & Settings packages are maintained.

Original team has forked this repository and working on Polkadot NFT marketplace - KodaDot and experimental NFT landscape - Metaprime.network

Feel free to join their Discord - KodaDot Polkadot NFT explorer & MetaPrime community

You can check products build thanks to these libraries

  • Polkadot NFT marketplace
  • metaprime.network

This is a collection of Vue.js libraries, utlities and classes that allows for easier use of the Polkadot JS infrastructure inside your apps.
